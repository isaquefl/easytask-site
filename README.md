# EasyTask

AI-powered personal finance control platform.

## Live App

**https://easytaskbetter.vercel.app**

## Features

- Dashboard with charts and metrics
- Debt tracking with automatic interest calculation
- Task management
- AI financial assistant (Groq / LLaMA 3.1)
- PIX payment integration (Mercado Pago)
- Secure auth with Better Auth

## Stack

Next.js | TypeScript | Prisma | PostgreSQL | Emotion | Groq AI
